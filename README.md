# Tressure-Island-Project
Treasure Island – Python Adventure Game

Treasure Island is a text-based adventure game developed using Python that allows users to navigate through a series of decisions in order to find hidden treasure. The program interacts with the player through the command line, presenting different scenarios where each choice determines the outcome of the game.

The game uses conditional statements (if, elif, else) to create a branching storyline where the player must choose the correct path to successfully reach the treasure. Incorrect decisions lead to different “Game Over” scenarios, making the experience interactive and engaging.

Key Concepts Used:
	•	Conditional statements (if, elif, else)
	•	User input handling using input()
	•	Nested decision logic
	•	String comparison and control flow
	•	Basic game logic implementation in Python

Game Flow:
	1.	The player starts at a crossroad and must choose a direction.
	2.	If the correct path is chosen, they reach a lake and must decide whether to wait or swim.
	3.	Reaching the island leads to a house with three colored doors.
	4.	Choosing the correct door leads to the treasure and winning the game.

This project is part of the 100 Days of Code – Python Bootcamp, designed to strengthen problem-solving skills and understanding of control flow in Python while creating an interactive command-line game.
